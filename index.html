FAQ

Frequently Asked Questions About Prediction Market Data & the Delphi API

Prediction market terminal, L1/L2/L3 order book data, cross-exchange feeds, and API access — everything you need to know.

What is Delphi Terminal?
Delphi Terminal is an aggregated prediction market data terminal providing institutional-grade L1, L2, and L3 market data across 7+ exchanges — including Kalshi, Polymarket, PredictIt, Limitless, Opinion Labs, Metaculus, and Manifold. It offers a web-based terminal interface and a comprehensive API for real-time and historical data access.
Which exchanges do you support?
Delphi aggregates data from 7+ prediction market exchanges: Kalshi, Polymarket, PredictIt, Limitless, Opinion Labs, Metaculus, and Manifold. We continuously expand coverage to provide the most comprehensive cross-exchange dataset available.
Is the data real-time?
Yes. Delphi provides real-time streaming data via WebSocket and REST endpoints. L1 top-of-book, L2 full order book depth, and L3 order-by-order data are all available with low-latency delivery.
Do you offer API access?
Yes. Delphi provides a comprehensive REST and WebSocket API supporting L1, L2, and L3 data feeds across all supported exchanges. Full documentation is available at docs.delphiterminal.co.
Who is Delphi for?
Delphi is built for traders, quantitative researchers, market makers, hedge funds, academic researchers, and developers who need institutional-grade prediction market data — whether for building trading algorithms, conducting research, or analyzing crowd-sourced signals.
Do you provide historical data?
Yes. Delphi maintains a growing historical dataset across all supported exchanges. Historical L1, L2, and L3 data is available via API for backtesting, model training, and academic research.
How is Delphi different from open-source tools?
Open-source tools typically cover a single exchange and require significant infrastructure work. Delphi aggregates 7+ exchanges into a single normalized feed with matched markets, full order book depth, and real-time streaming — built for institutional use without the setup overhead.
How do I get access?
You can sign up at delphiterminal.co/checkout. Plans range from a free tier to institutional access. For custom enterprise arrangements, reach out via the contact form below.
